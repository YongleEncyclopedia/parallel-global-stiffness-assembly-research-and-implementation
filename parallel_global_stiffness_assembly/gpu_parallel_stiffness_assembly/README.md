# Parallel Global Stiffness Assembly Workspace

This workspace is organized into three purpose-based areas:

- `implementation/gpu_parallel_stiffness_assembly/` - CUDA/C++ benchmark implementation, build entry, tests, scripts, measured results, and project-local documentation.
- `literature/papers/` - primary research papers, renamed with author/year/topic slugs.
- `literature/reviews/` - Markdown and PDF literature-review reports.
- `project_docs/` - project planning, environment setup, testing/visualization design, and architecture notes that support the implementation.

The cleanup removed generated build products, editor caches, local agent/tool state, duplicate PDFs, old archive ZIPs, empty placeholder files, and superseded report copies. The project is not currently inside a Git worktree, so this README also serves as a light inventory of the cleaned layout.
